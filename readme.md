# Lange970

Luxury mechanical-inspired Garmin Connect IQ watch face for the Forerunner 970.

## Goal

Build an original analog Garmin watch face inspired by high-end perpetual calendar design language, with a charcoal dial, rose-gold accents, off-center time dial, month ring, battery arc, and Garmin complications.

## Target Device

- Garmin Forerunner 970
- 454 × 454 AMOLED display
- Connect IQ SDK 9.2.x

## Current Status

Initial Connect IQ project created and building.

## Planned Features

- Charcoal luxury dial
- Rose-gold chapter ring
- Off-center analog time dial
- Large framed date window
- Physical-style month ring
- Battery arc from roughly 7:15 to 2:00
- Moonphase/seconds complication
- Steps and heart-rate complications
- Always-On Display optimization
